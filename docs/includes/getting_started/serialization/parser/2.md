=== "AIOKafka"
    ```python
    from types import Callable, Awaitable
    from faststream.kafka import ConsumerRecord, KafkaMessage

    async def parser(
        msg: ConsumerRecord,
        original_parser: Callable[[ConsumerRecord], Awaitable[KafkaMessage]],
    ) -> KafkaMessage:
        return await original_parser(msg)
    ```
